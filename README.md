**Zento
**

**Modern React e-commerce product listing landing page built with React, TypeScript, Vite, and Tailwind CSS.
**

**Overview**

Zento is a modern and responsive e-commerce frontend project focused on clean UI, product exploration, and smooth user experience.
The project showcases:

Product listings
Search functionality
Responsive layouts
Modern component architecture
API integration
Reusable UI components

This project is designed to strengthen frontend development skills using real-world React patterns and scalable folder structures.

**Features**

1. Product search functionality
2. Dynamic product listing
3. Fully responsive design
4. Fast development with Vite
5. Styled with Tailwind CSS
6. Reusable React components
7. API-based product fetching
8. Modern clean UI
   
**Tech Stack**

**Frontend**:
React
TypeScript
Vite
Tailwind CSS
APIs
Fake Store API
